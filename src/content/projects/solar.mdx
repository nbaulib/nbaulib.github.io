## Challenges
- Some keys did not update color correctly on press  
  - Fix: refined event handling for consistent state updates  

- Could not achieve desired material look using `metalness()`  
  - Fix: switched to solid colors  

- Sun wave recoil felt unnatural after key release  
  - Fix: implemented a separate key release function to control animation reset  

## Things we're proud of 
- Smooth expanding and collapsing sun-wave interaction  
- Rotating text  
- Clipping the audio right
